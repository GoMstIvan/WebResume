# Role
請扮演一位專業的 UI/UX 設計師兼前端工程師。

# Task
請使用 HTML 和 CSS (建議使用 Tailwind CSS via CDN 以方便樣式調整)，製作一份單頁的、專業的履歷模板。設計風格需嚴格參照以下的視覺需求。

# Visual Style & Layout Requirements (視覺與佈局需求)
1.  **版面結構 (Layout):**
    - 採用 **左右雙欄式佈局 (Two-Column Layout)**。
    - **左側欄 (寬度約 30%):** 用於放置個人頭像、基礎信息、聯繫方式、證書獎勵、QR Code。
    - **右側主欄 (寬度約 70%):** 用於放置頂部 Header (姓名/職稱)、教育背景、工作經歷、校內實踐、自我評價。

2.  **配色方案 (Color Palette):**
    - **主色調:** 莫蘭迪綠 / 灰綠色 (Sage Green / Muted Teal，例如 hex code #8CA9A3)。
    - **背景:** 乾淨的白色。
    - **文字:** 深灰色，確保易讀性。

3.  **頂部 Header 設計 (關鍵):**
    - 頂部需清晰展示「姓名」與「求職意向」。
    - **裝飾元素:** 在頁面的 **右上角** 設計一個由主色調構成的 **幾何斜紋線條 (Diagonal stripes)** 裝飾，以還原參考圖的設計感。

4.  **區塊樣式:**
    - **左側欄標題:** 使用「旗標式」或背景色塊樣式來突顯標題 (例如「基礎信息」、「聯繫方式」)，背景色使用主色調，文字反白。
    - **右側欄標題:** 簡約現代風格，底部可帶有細裝飾線。

5.  **內容排版:**
    - **工作經歷:** 採用列表或時間軸格式，左側顯示時間區間 (如 2013.09 -- 2017.06)，右側顯示公司名稱、職位與詳細工作描述。
    - **照片:** 左上角預留圓形或圓角矩形的圖片區塊。

# Output Requirement (輸出要求)
- 提供完整的單一 HTML 檔案原始碼。
- 內容請全部使用 **通用佔位符 (Placeholders)** (例如：「[您的姓名]」、「[求職意向]」、「[公司名稱]」、「[工作描述...]」)，方便我後續替換文字。
- 確保 CSS 樣式在瀏覽器中預覽整齊，並適合 A4 紙張列印。
